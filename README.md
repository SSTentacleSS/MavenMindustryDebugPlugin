# **MindustryDebugPlugin**

Maven plugin for debugging Mindustry

# Installing

add this to `pom.xml` file
```xml
<repositories>
    <!-- ... -->

    <repository>
        <id>jitpack.io</id>
        <url>https://jitpack.io</url>
    </repository>
</repositories>

<!-- ... -->

<build>
    <plugins>
        <plugin>
            <groupId>sample.plugin</groupId>
            <artifactId></artifactId>
            <version>1.0.0</version>
        </plugin>
    </plugins>
</build>
```
